Les codes sources suivants sont les exemples utilisés lors du webinaire du 26 juin 2021.

Vous devez installer la version sous licence ou d'évaluation des différentes librairies utilisées pour les compiler et voir ce que ça donne sur votre ordinateur.

Pour voir ou revoir ce webinaire, connectez-vous sur https://serialstreameur.fr/webinaire-20210624.php
